# 蒸氣龐克空氣曲棍球 Steampunk Edition v1.1

這版加入進階震動回饋 Haptic Engine。

## 新增震動手感

- 點擊球桌：短震動
- 撞牆：輕微短震
- 普通擊球：中等短震
- Perfect Hit：多段強震
- 進球：長段節奏震動
- 倒數：短節奏震動
- GO：啟動震動
- 勝利：勝利節奏震動
- 失敗：低沉長震

## 原本蒸氣龐克風格保留

- 黃銅 / 古銅色系
- 齒輪球桌裝飾
- 機械球拍
- 鉚釘圓環
- 蒸氣煙霧粒子
- 鍋爐能量光暈
- 4 種主題：Steampunk / Copper / Royal Brass / Coal Engine

## 注意

震動功能依賴瀏覽器與裝置支援。  
若手機瀏覽器不支援，遊戲仍可正常遊玩，只是不會震動。
